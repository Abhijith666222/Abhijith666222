## 🧑‍💻 About Me

I'm a **Double Degree** student at **NTU Singapore** (Business & Computing) with a focus on applied AI, full-stack development, and systems engineering. I build things that work — from LLM-driven agentic systems to multimodal accessibility platforms.

Currently an **Undergraduate Researcher** working on Residue Number Systems for Machine Learning under NTU's URECA programme. Previously shipped production AI systems at **Crayon Data** as an AI Engineer & Entrepreneur in Residence, and built internal tools used by 500+ personnel during my time with the **Singapore Armed Forces**.

I'm drawn to deterministic, explainable systems over black-box automation. If it ships, it should make sense.
